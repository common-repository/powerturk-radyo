== Description ==

Bu eklenti sayesinde Wordpress blogunuzda PowerTurk Player kurulmaktadır.

== Installation ==

1. Wordpress Admin panelinde, EKLENTİLER > Yeni EKLE ve  'PowerTurk' olarak arama yapın.

2. 'Yükle' Tıklayın.

3. Wordpress Admin içinde , Görünüm > Bileşenler, içinde  'PowerTurk Radyo’ eklentisini istediğiniz bölüme sürükleyip yerleştirin ve kaydedin..
